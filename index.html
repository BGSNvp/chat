``` --- ### **Step 2: Update Your Firestore Security Rules** Here are the new security rules. They include a new section for an `admins` collection. This is what allows the new code to securely check for admin status. Please replace the rules in the `"Firestore Security Rules for Moderated Chat"` document with this new version. ```javascript rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { // Rules for the 'admins' collection // This collection holds a document for each admin. match /admins/{userId} { // Only the user whose ID matches the document ID can read it. // This is how the app securely checks for admin status. allow read: if request.auth.uid == userId; // No one can write to this collection from the app for security. // Admins must be added manually in the Firebase console. allow write: if false; } // Path for all public app data match /artifacts/{appId}/public/data/{document=**} { // By default, deny all reads and writes unless specified below. allow read, write: if false; } // Rules for the 'users' collection (where handles are stored) match /artifacts/{appId}/public/data/users/{userId} { // Anyone can read any user's handle. allow read: if true; // A user can only create or update their own handle. allow write: if request.auth.uid == userId; } // Rules for the 'comments' collection match /artifacts/{appId}/public/data/comments/{commentId} { // Anyone can read comments that have been approved. // The admin can read any comment, including pending ones. allow read: if resource.data.status == 'approved' || (get(/databases/$(database)/documents/admins/$(request.auth.uid))).data.isAdmin == true; // Any authenticated user can create a new comment, but it must be // marked as 'pending' and owned by them. allow create: if request.auth != null && request.resource.data.status == 'pending' && request.resource.data.author == request.auth.uid; // Only the admin can update a comment (to approve it). allow update: if (get(/databases/$(database)/documents/admins/$(request.auth.uid))).data.isAdmin == true; // Only the admin can delete a comment. allow delete: if (get(/databases/$(database)/documents/admins/$(request.auth.uid))).data.isAdmin == true; } } }
